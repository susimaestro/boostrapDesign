Diction Wize for mobile
June 2020
UX/UI Designer

The goal for this project was to make a dictionary that was easier to use and beautiful to look at. 9/10 users said the dictionary would be a more engaging experience if the content was displayed in a more visually pleasing manner.

Tiffin Café E-commerce website
April 2020
UX/UI Designer
Tiffin Café wanted to increase their online presence with a website that would be an authentic representation of them and their family run business.

"Honest", "Authentic", "A home a way from home", "Healthy", "Friendly" were the main keywords that kept popping up when users were asked to describe Tiffin Café and these became the keywords we were striving for with the design.
